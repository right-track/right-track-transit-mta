Right Track Transit: MTA

A Right Track Transit Agency implementation for the Metropolitan Transportation Authority

Metropolitan Transportation Authority

GitHub repo: right-track/right-track-transit-mta

This module is an implementation of a RightTrackTransitAgency used to create a real-time Transit Feed for the MTA in New York City, which is used in the various Right Track Projects.

Documentation

Documentation about the RightTrackTransitAgency class and TransitFeed classes can be found in the right-track-core project and online at https://docs.righttrack.io/right-track-core.

Usage

This example builds a TransitFeed for the MTA:

const mta = require('right-track-transit-mta');

// Optional: Override default agency properties
mta.readConfig('/path/to/agency_local.json');

mta.loadFeed(function(err, feed) {
  if ( !err ) {
    console.log(JSON.stringify(feed, null, 2));
  }
});
